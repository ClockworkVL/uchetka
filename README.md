# Учет продаж

Локальное приложение для учета продаж по сменам. Можно открыть как обычный `index.html` или установить как Windows-программу.

## Запуск

Для быстрого запуска откройте файл `index.html` в браузере.

Для запуска как программы используйте установщик из GitHub Releases.

## Установка на Windows

Скачайте установщик `Uchetka-Setup-*.exe` из раздела Releases на GitHub и запустите его.

Установленная программа проверяет обновления в GitHub Releases при запуске. Когда новая версия будет опубликована, она скачает обновление и предложит перезапустить программу.

## Что умеет

- Добавлять несколько товаров в одну продажу.
- Указывать способ оплаты: `экв`, `нал` или `пер`.
- Запоминать введенные товары и последнюю цену.
- Выбирать сохраненный товар из списка.
- Закрывать смену и сохранять итог за день.
- Смотреть отчеты за день, неделю, месяц и все смены с разбивкой по оплате.

Данные хранятся локально на этом компьютере.
