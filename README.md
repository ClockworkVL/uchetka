# Учет продаж

Локальное приложение для учета продаж по сменам. Можно открыть как обычный `index.html` или установить как Windows-программу.

GitHub: https://github.com/ClockworkVL/uchetka

## Запуск

Для быстрого запуска откройте файл `index.html` в браузере.

Для запуска как программы используйте установщик из GitHub Releases.

## Установка на Windows

Скачайте установщик `Uchetka-Setup-*.exe` из раздела Releases и запустите его:

https://github.com/ClockworkVL/uchetka/releases/latest

Установленная программа проверяет обновления в GitHub Releases при запуске. Когда новая версия будет опубликована, она скачает обновление и предложит перезапустить программу.
Также обновление можно проверить и установить вручную из блока `Настройки` кнопками `Проверить обновление` и `Обновить`.

Продажи, товары, настройки и закрытые смены сохраняются в профиле приложения Windows и не удаляются при обновлении. При первом запуске новой версии старые данные из прежнего хранилища автоматически переносятся в постоянный файл данных.

## Выпуск новой версии

1. Запустите `npm run dist` или `npm run publish`.
2. Скрипт автоматически увеличит patch-версию в `package.json` и `package-lock.json`.
3. Сделайте коммит версии и отправьте его на GitHub.
4. Для GitHub Actions создайте и отправьте тег с этой версией, например `v1.0.11`.

GitHub Actions соберет установщик без повторного увеличения версии. Установленная программа увидит опубликованный релиз при следующем запуске.

## Что умеет

- Добавлять несколько товаров в одну продажу.
- Указывать способ оплаты: `экв`, `нал` или `пер`.
- Запоминать введенные товары и последнюю цену.
- Выбирать сохраненный товар из списка.
- Вести базу товаров на отдельной вкладке и структурировать товары по категориям.
- Выбирать существующие категории и подкатегории при добавлении товара или создавать новые.
- Добавлять подкатегории внутри категорий товаров.
- Редактировать сохраненный товар: название, цену, категорию и подкатегорию.
- Раскрывать категории товаров и продажи по стрелке.
- Закрывать смену и сохранять итог за день.
- Смотреть отчеты на отдельной вкладке за день, неделю, месяц и все смены с разбивкой по оплате.
- Печатать товарный чек по выбранной продаже из текущей смены.
- Выбирать светлую или темную тему.
- Сохранять название компании для шапки приложения и товарного чека.
- Проверять и устанавливать обновления из настроек.

Данные хранятся локально на этом компьютере.
